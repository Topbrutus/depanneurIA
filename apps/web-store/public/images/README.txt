=== DOSSIER PHOTOS PRODUITS ===

Placez vos photos ici. Format recommandé : JPG ou WebP, 300x300px.

Nommez vos fichiers comme suit :
  lays-ketchup.jpg
  pepsi-355ml.jpg
  etc.

Ensuite dans src/data.ts, mettez le chemin :
  image: '/images/lays-ketchup.jpg'

Pour compresser vos photos automatiquement :
  bash compress-images.sh
