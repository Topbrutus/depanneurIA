# Comment déposer ce pack dans GitHub

## Option 1 — en local avec git

1. Extraire le zip
2. Copier le contenu dans ton dossier local du dépôt `depaneurIA`
3. Vérifier les fichiers
4. Faire :

```bash
git add .
git commit -m "chore: initialize GitHub tracking pack"
git push
```

## Option 2 — via l’interface GitHub

1. Ouvrir le dépôt
2. Ajouter les fichiers un par un, ou glisser-déposer les fichiers si tu travailles depuis un dossier local
3. Respecter l’arborescence `.github/`, `docs/`, etc.

## Premier commit conseillé

`chore: initialize GitHub tracking pack`
